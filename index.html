Hacktoberfest?

Es la celebracion del software libre (open source). Donde se invita a todas las personas a contribuir en algun proyecto hosteado en GitHub durante el mes de Octubre.

Reglas

Para obtener la playera conmemorativa debes completar 5 pull request durante el mes de Octubre. Los puedes hacer en cualquier repo publico de GitHub.
